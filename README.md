# Python3とFlaskを使ったQRコード生成アプリ

## 開発環境
- サーバー
    - Raspberry Pi3 B
    - Raspbian 10
    - Python 3.7.3
    - Flask 1.0.2
- クライアント
    - Windows10
    - Visual Studio Code Remote Development

## What?
PythonのQRコード生成ライブラリの```qrcode```を使用して、QRコードを生成するWebアプリ。

ライブラリの詳細 -> <a href="https://github.com/lincolnloop/python-qrcode">python-qrcode</a>
